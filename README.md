# Simple Interest Calculator

## Description
This project is a simple interest calculator that computes interest based on principal, rate, and time.

## Formula
Simple Interest = (P × R × T) / 100

Where:
- P = Principal amount
- R = Rate of interest (per year)
- T = Time (in years)

## How to Run

### Example in Python:
```python
P = float(input("Enter principal: "))
R = float(input("Enter rate: "))
T = float(input("Enter time: "))

SI = (P * R * T) / 100
print("Simple Interest is:", SI)
